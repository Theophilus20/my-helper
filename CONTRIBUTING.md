# Contributing to My Helper

Thank you for helping improve My Helper.

My Helper is designed to help people know how to use ChatGPT and Codex. Contributions should keep the experience encouraging, understandable, accessible, and respectful.

Before opening an issue, search existing issues to see whether the topic has already been reported.

For a bug report, explain what you expected, what happened instead, and the steps needed to reproduce the problem.

For a feature request, explain the user problem first. Describe how the idea would help beginners, older adults, students, developers, or users who need accessibility support.

Before opening a pull request, test the change in Chrome on ChatGPT. Keep the change focused. Do not include API keys, `.env` files, private user data, or screenshots containing sensitive information.

All contributors are expected to follow the Code of Conduct.
