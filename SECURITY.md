# Security Policy

## Supported Version

The latest released version of My Helper receives security updates.

## Reporting a Vulnerability

Do not create a public issue for a possible security vulnerability.

Use the Report a vulnerability button in the Security section of this GitHub repository. Include a clear description of the issue, steps to reproduce it, the possible impact, and any suggestions that may help resolve it.

Please allow time for the report to be reviewed and fixed before sharing details publicly.

## Security Principles

My Helper does not include API keys in browser extension code.

API keys must remain in environment variables on the backend.

My Helper does not automatically send ChatGPT messages, press Send, upload files, or change user settings.

Visual analysis is optional and must be enabled by the user before a visible browser image is captured.
