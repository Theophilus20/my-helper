# Code of Conduct

## Our Pledge

We want My Helper to be a welcoming and respectful project for everyone.

## Expected Behavior

Use kind and constructive language.

Respect different backgrounds, skill levels, cultures, identities, and experiences.

Give feedback about ideas and code without attacking people.

Help create a learning environment where beginners feel comfortable asking questions.

## Unacceptable Behavior

Harassment, discrimination, threats, personal attacks, hateful language, and sharing private information without permission are not allowed.

## Reporting

Report concerns privately through the repository contact details or GitHub private reporting tools.

All reports will be reviewed respectfully and privately.
